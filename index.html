Diferenças entre tags de bloco e tags em linha:

Tags de Bloco:
Bloco: Tags de bloco ocupam toda a largura disponível e criam uma nova linha antes e depois do elemento. Elas são usadas para criar seções distintas na página.
Tags em Linha:
Tags em linha ocupam apenas o espaço necessário para o seu conteúdo e não criam uma nova linha antes ou depois do elemento. Elas são usadas para estilizar ou formatar pequenas partes do conteúdo.
